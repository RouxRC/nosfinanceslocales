NosFinanceLocales.fr a pour objectif de présenter une petite partie des données comptables des communes françaises : nous ne présentons en effet que 7 cartes sur plus de 60 variables disponibles. N'hésitez donc pas à récupérer l'ensemble des données ainsi que les descriptions des variables sur NosDonnées.fr.

Quelles données ?

  • Les données financières des communes : extraites du site des collectivités locales et disponible en format csv sur le site NosDonnées.fr en Open Data !
  • Les données géographiques des communes : IGN GEOFLA

Quelle licence ?

Toutes les données de ce site sont sous la licence LO.

Comment ?

  • La cartographie avec mapnik et CartoCSS
  • Le site web dont le code est dispo sur github avec coté serveur Pyramid
    • et coté client AngularJS
  • Le crawler qui a permis de récolter toutes les données financières : code disponible sur github également
